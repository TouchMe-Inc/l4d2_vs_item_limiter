# About vs_item_limiter
The plugin limits the number of items that can be on the map.

## ConVars
| ConVar                    | Value       | Description                                                                           |
| ------------------------- | ----------- | ------------------------------------------------------------------------------------- |
| sm_item_tracking          | 0           | Keep item spawns the same on both rounds.                                             |
| sm_item_pain_pills_limit  | -1          | Limits the number of pain_pills on each map. -1: no limit; >=0: limit to cvar value.  |
| sm_item_adrenaline_limit  | -1          | Limits the number of adrenaline on each map. -1: no limit; >=0: limit to cvar value.  |
| sm_item_pipe_bomb_limit   | -1          | Limits the number of pipe_bomb on each map. -1: no limit; >=0: limit to cvar value.   |
| sm_item_molotov_limit     | -1          | Limits the number of molotov on each map. -1: no limit; >=0: limit to cvar value.     |
| sm_item_vomitjar_limit    | -1          | Limits the number of vomitjar on each map. -1: no limit; >=0: limit to cvar value.    |
